# PalindromeCheckerApp

## Project Overview

PalindromeCheckerApp is a Java-based application that demonstrates
multiple approaches to check whether a given string is a palindrome. A
palindrome is a word, phrase, or sequence that reads the same forward
and backward, such as **madam**, **level**, or **racecar**.

This project is designed to demonstrate **Object-Oriented Programming
(OOP)** concepts and **Data Structures** in Java while implementing
different palindrome checking techniques.

------------------------------------------------------------------------

## Features

-   Multiple palindrome checking algorithms
-   Demonstrates Data Structures such as Stack, Queue, Deque, and Linked
    List
-   Demonstrates Object-Oriented Programming concepts
-   Case-insensitive palindrome checking
-   Recursive palindrome checking
-   Performance comparison of algorithms

## Description of Files

  --------------------------------------------------------------------------------
  File                         Description
  ---------------------------- ---------------------------------------------------
  PalindromeCheckerApp.java    Main entry point of the application

  HardcodedPalindrome.java     Basic palindrome check using reverse method

  CharacterArray.java          Uses character array comparison

  CaseInsensitive.java         Checks palindrome ignoring spaces and letter case

  StringReverse.java           Manual string reverse implementation

  StackBased.java              Uses Stack data structure

  QueueStack.java              Uses both Queue and Stack

  DequeueBased.java            Uses Deque (double-ended queue)

  LinkedListBased.java         Uses Linked List reversal technique

  Recursive.java               Recursive palindrome checking

  ObjectOriented.java          Demonstrates OOP implementation

  StrategyPattern.java         Uses Strategy Design Pattern

  PerformanceComparison.java   Compares execution time of algorithms
